# Bikcraft

Desenvolvimento do projeto do curso ``Web Design Completo`` da plataforma de cursos [Origamid](https://www.origamid.com/).

O projeto será desenvolvido desde o layout até a codificação, passando pelas seguintes etapas:

- Design do Projeto
- Utilização do AdobeXD
- Criação de Wireframe
- Protótipo
- UX e UI Design
- Desenvolvimento
